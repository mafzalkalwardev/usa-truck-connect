# Contributing to Mighty Trucking

Thank you for your interest in contributing! This is a static affiliate marketing website — contributions are welcome for UI improvements, accessibility, documentation, and bug fixes.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/mighty-trucking.git`
3. Serve locally: `npm run serve` or `npx serve .`
4. Create a branch: `git checkout -b feature/your-feature`

## Development Guidelines

- Match existing code style (semantic HTML, CSS custom properties, vanilla JS)
- Use relative paths for all internal links (GitHub Pages compatibility)
- Test on mobile viewport (375px) and desktop (1440px)
- Do not commit secrets, API keys, or personal email addresses
- Google trademarks: follow [Google Brand Permissions](https://about.google/brand-resource-center/)

## Pull Request Process

1. Update README if you add pages or features
2. Run `npm run screenshots` after visual changes (requires Playwright)
3. Describe changes clearly in the PR title and body
4. One feature per PR when possible

## Reporting Issues

Open an issue with:
- Page URL or file name
- Browser and device
- Steps to reproduce
- Expected vs actual behavior
