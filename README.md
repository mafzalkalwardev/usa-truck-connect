# USA Truck Connect

Professional affiliate marketing website for Google Workspace + Google Voice sign-up rewards.

## Pages

- **Home** — Hero, live stats, trust badges
- **About** — Mission and compliance
- **Offer** — $40 reward details and FAQ
- **How It Works** — Interactive wizard + animated walkthrough
- **Testimonials** — Social proof
- **Join** — Affiliate signup form
- **Thank You** — Post-submission confirmation
- **Privacy / Terms** — Legal pages

## Local Development

Open any HTML file in a browser, or serve locally:

```bash
npx serve .
```

## Deployment (GitHub Pages)

1. Push to `main` branch
2. Enable GitHub Pages: Settings → Pages → Source: `main` / root
3. Live at: [https://mafzalkalwardev.github.io/usa-truck-connect/](https://mafzalkalwardev.github.io/usa-truck-connect/)

## Custom Domain

Add a `CNAME` file with your domain (e.g. `usatruckconnect.co`) and configure DNS:

```
CNAME  www  <username>.github.io
```

## Form Handling

Signup form uses [FormSubmit.co](https://formsubmit.co) — no backend required.

## Tech Stack

- Static HTML / CSS / JavaScript
- [AOS](https://michalsnik.github.io/aos/) for scroll animations
- Font Awesome icons
